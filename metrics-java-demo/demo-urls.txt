## To get a customer by id

GET http://localhost:8080/metrics-java-demo/api/customer/1

## To create a customer

POST http://localhost:8080/metrics-java-demo/api/customer

<customer>
	<id>1</id>
	<firstName>John</firstName>
	<lastName>Doe</lastName>
</customer>

## To check the external HTTP service stub is deployed

GET http://localhost:8080/http-service-stub/myresource/address

GET http://localhost:8080/metrics-java-demo/operations
